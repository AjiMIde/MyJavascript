### 3.5 条件与循环

* 3.5.1 循环

##### 3.5.1 循环
* 原则上，使用 for () {} 来执行处理循环业务，而非 while
* 不在 for 循环中使用多余表达式
* 原则上，不使用倒序来处理循环，因为可读性差
```js
// bad
let i = 0
let len = 10
while (i < 10) {} 

for (let i = 10, j = len -i; i < len; i--) {}

// good
for (let i = 0; i < len; i++) {
  let j = len - i
}
```

##### 3.5.2 switch 分支判断
* switch 判断一定要自带 default
* break 字样一定要带上
* 原则上，分支条件一定要相互独立（避免同个判断符合多个条件）

```js
switch (i) {
  case i < 10:
    break
  case i < 5:       // 错误
    break
  default:
} 
```

##### 3.5.3 条件判断
* 使用 === 而非 ==
* 使用 !== 而非 !=
* 由于 0 == false, '' == false，故这些判断有时会造成业务上的混淆
```js
let a
a == null           // true      
a === null          // false
a === undefined     // true
```
